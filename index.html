Moonlight

By Tembee2

How to Use

After setting up all modules as described in the README, use the chat trigger (/moonlight) to open the moderation UI (for admins, mods or others only).

Commands

Ban (Temp)

Temporarily bans a player. Requires username, reason, private reason, and duration (in hours).

Ban (Perm)

Permanently bans a player. Requires username, reason, and private reason.

Unban

Removes a player from the ban list.

Kick

Kicks a player from the server. Requires username and reason.

Invisible

Makes a player invisible.

Noclip

Allows a player to walk through objects.

Teleport

Teleports a player to a specified location.

Shutdown

Shuts down the server. Use with caution.

Kill

Kills a player's character.

Walkspeed

Sets a player's walkspeed. With a slider between 0–255.
